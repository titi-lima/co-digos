CRUD

Em backend, CRUD (Create, Read, Update, Delete) é o nome dado ao conjunto de processos essenciais para a criação e manutenção de um banco de dados. Eles indicam, respectivamente, as funções de inserção, leitura, atualização e deleção de um dado do banco. Nessa fase, usaremos a API do Github para simularmos processos de leitura de dados.
